# Lambda
---
### What is AWS Lambda

- Function as a Service (FaaS)
- Serverless
- Automatic scaling & pay as you go

---
### What can it Do?

- Runs your code
- Event driven
    - Manually triggered
    - S3
    - Cron (EventBridge Rule)

---
### Whats Supported
- Node.js
- Python
- Ruby
- Java
- .NET
- Container Images

---
### Other Features
- Monitoring metrics
- Memory & Timeout
- Integrates with AWS IAM
- Environment variables
- Attach to VPC to allow connectivity to private resources
- Concurrency
